- **Focus:** shipment workflows, document handling, dashboard visibility
- **Stack:** React, TypeScript, Node.js, Express, MySQL
- **Live demo:** https://expor-track-ai.vercel.app

### Site Surveyor Compass
Flutter-based field utility app for GPS workflows, measurement, waypoint management, and on-site reporting.
- **Focus:** field usability, accuracy, and mobile-first practicality
- **Stack:** Flutter, Dart, Provider, Hive, Firebase

### Planora
Calendar and admin platform with recurrence, ICS import/export, and structured event management.

- **Stack:** React, TypeScript, Node.js, Express, Prisma, PostgreSQL
- **Strength:** monorepo setup with authentication, admin tools, and test coverage

- **Stack:** FastAPI, Python, JavaScript, HTML/CSS
